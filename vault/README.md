# Locked Vault

This directory is reserved for working-source material: TXT files, Markdown transcriptions, neurology notes, correspondence extracts, document inventories, and other reference material used to build the public exhibit pages.

## Important security note

This repository is currently public. A folder inside a public GitHub repository is not truly locked. Anything committed here can be viewed by anyone with repository access, even if the site does not link to it.

For genuinely confidential material, use one of these approaches instead:

1. Keep the source documents outside this public repository.
2. Use a separate private repository for the vault.
3. Commit only redacted excerpts here.
4. Encrypt sensitive files before committing them.

## Vault rule

Public pages should not link directly to unredacted source material. They should link only to controlled exhibit summaries or redacted excerpts.

## Intended structure

```text
vault/
  README.md
  manifest.md
  neurology-notes/

```

## Editorial use

The vault is for reference and drafting. The public site should remain pedagogical: exhibit cards first, controlled excerpts second, full documents only where appropriate and redacted.

# Vault Manifest

This manifest indexes source materials intended for private reference, controlled transcription, and later exhibit drafting.

## Status labels

- `pending` — not yet added
- `redacted` — safe public excerpt prepared
- `private` — should not be published in a public repository
- `public-summary` — represented only by an exhibit summary

## Core source categories

| Category | Intended folder | Status | Notes |
|---|---|---:|---|
| Neurology notes | `vault/neurology-notes/` | pending | TXT or Markdown working versions only; redact identifiers before public use. |
| GP referral | `vault/referrals/` | pending | Use controlled excerpts for exhibit support. |
| Discharge summary | `vault/discharge/` | pending | Core closure artefact; preserve sequence language carefully. |
| LTI certification | `vault/certification/` | pending | Reliance artefact; consider redacted transcription. |

| Correspondence | `vault/correspondence/` | pending | Use only relevant excerpts. |

| St Vincent's / genetic confirmation | `vault/genetic-confirmation/` | pending | Causal completion artefact; redaction required. |

## Working rule

Do not place unredacted source material in this public repository. Use this manifest to track what exists, what is safe, and what remains private.

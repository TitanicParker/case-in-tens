# Case in Tens

A phone-first GitHub Pages documentary archive for teaching the case through memorable Top 10s, exhibit cards, and concise pedagogical pages.

## Core frame

Finality must follow inquiry. It must not precede it.

## Site texture

The design is a restrained hall-of-fame / case-museum interface: dark courtroom background, brass plaque accents, document cards, and short exhibit captions.

## Pages

- `index.html` — Hall landing page
- `start-here.html` — five-minute guided orientation
- `top-10s.html` — sharp list-based teaching engine
- `essays.html` — index of the twelve essays
- `exhibits.html` — artefact gallery
- `timeline.html` — hinge chronology
- `glossary.html` — teaching definitions

## Publication

Enable GitHub Pages from the repository settings, using the `main` branch and root folder.

## Editorial rule

The archive should argue from the record outward, not from allegation inward.
